* ## Proje 1
  
   **[22,27,16,2,18,6]** &rarr;  Insertion Sort

   1. Yukarıda verilen dizinin sort türüne göre aşamalarını yazınız.
 
       **Cevap :**

      **[2,27,16,22,18,6]**
      
      **[2,6,16,22,18,27]**
      
      **[2,6,16,22,18,27]**
      
      **[2,6,16,18,22,27]**
      
      **[2,6,16,18,22,27]**

    2. Big-O Gösterimini yazınız.
 
       **Cevap :**
```math
 n+ (n-1)+(n-2)+...+1 = {n.(n+1) \over 2} = { n^2 + n \over 2}
``` 
    
      olduğundan big-O gösterimi 
      
  ```math
O(n^2)
```
